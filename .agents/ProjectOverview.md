# Project Overview

## プロジェクト概要
本作は「Slay the Spire」のローグライクデッキビルディング要素と、「Epic War 3」のような2D大群戦闘を融合させた、新しい形のストラテジーゲームです。

## コアメカニクス（最新アップデート: リアルタイム・一括召喚システム）
本作の最大の特徴は、**「時間を止めず、常に前線が動き続ける中で、次のターンの援軍を展開予約する」**点にあります。リアルタイムの緊張感と、カードゲームの思考・予測の楽しさを両立しています。

### ⏳ リアルタイム・ウェーブ召喚制
- **時間は常に進み続ける**（時止めによるポーズはない）。
- **20秒ごとの「召喚サイクル」**でゲームが進行する（ゆったりとした思考を可能に）。
- サイクル開始時に手札（カード）が新しく配られ、マナが **5** にリセットされる（※持ち越し不可。余った分は消滅し、毎ターン5からのスタートとなる）。
- **予約召喚（ゴースト配置）**:
  - プレイヤーはリアルタイムに進行する戦闘を眺めながら、手札のユニットカードを戦場の「自陣側・または最前線の味方より後ろ」にドラッグ＆ドロップで配置（予約）する。
  - 配置した瞬間マナを消費し、そこに「半透明のゴースト（出撃待機中）」が現れる。
- **一括実体化（スポーン）**:
  - 20秒のサイクルタイマーが「0」になった瞬間、配置予約していた全てのゴーストが一斉に実体化し、怒涛の勢いで前線に突撃を開始する！
  - 同時に手札がリセット(破棄)され、新たな手札がドローされ、次の20秒サイクルが即座に始まる。
- **スペルの即時発動**:
  - 魔法（スペル）カードのみは例外で、20秒のタイマーを待たずに、手札から戦場へドロップした瞬間に**即時発動**し、前線を直接支援できる。

## 開発の状況と経緯
- **フェーズ1**: 基本的な戦闘モックの完成。キャラの個性（振りかぶりやノックバック等）、重量感のある戦闘テンポを確認。
- **フェーズ1.5 (現在)**: 「時止めフェーズ制」からより中毒性の高い「リアルタイム・ウェーブ召喚（一括配置予約制）」へのゲームコア大改修。プレイヤーは先を見据えた陣取りとマナリソース管理を同時に行う。
- **フェーズ2 (Next)**: StS風のマップルート選択・戦闘後のカード獲得/レリック取得のローグライクサイクルの実装。

## 重要なゲームバランス・デザイン方針
- **「高いやつ出せば勝ち」禁止**: 全ユニットに致命的な弱点があり、「組み合わせ」でのみ強くなる設計。
- **「祈る時間」の創出**: 援軍が到着（実体化）するまでの数秒間、いかに今の前線が持ちこたえれるかを応援するドラマチックなプレイ体験。
- **Juice（触り心地）の重視**: ノックバック、ヒットストップ、溜め攻撃、そして「援軍の一斉突撃時のドーパミン大放出」など、見た目と音の気持ちよさを最優先する。
- **操作の簡略化**: 「忙しいアクション」ではなく、「どこに・誰を置くか」というマクロな判断に集中させる。
